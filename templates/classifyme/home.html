Tentang Aplikasi

ClassifyMe! adalah aplikasi yang dibuat khusus untuk mengklasifikasikan resensi buku ke dalam resensi positif dan negatif dengan menggunakan teknik pre-processing pada text mining.


{% load static %}

Aplikasi ini dibuat oleh Adinda Octadia Putri, mahasiswi Universitas Gunadarma angkatan 2013 sebagai syarat dalam mencapai gelar Sarjana Strata Satu (S1).

Petunjuk Penggunaan

  1. Untuk menggunakan aplikasi, klik tombol "Mulai" yang ada pada bagian paling bawah halaman ini. Jika tombol tidak tertera maka Anda harus melakukan login terlebih dahulu di Halaman Administrasi.
  2. Pada tabel "CLASSIFYME" pilih "+Add" pada baris "Bukus", lalu masukkan data sesuai dengan kolom yang sudah tersedia.
  3. File data yang dapat diproses adalah file berformat .csv.
  4. Kembali ke Halaman Klasifikasi dan klik tombol "Mulai".
  5. Pilih buku yang akan diproses.
  6. Klik tombol "Proses" di bagian bawah halaman untuk melakukan proses klasifikasi.
  7. Setelah hasil klasifikasi ditampilkan, klik tombol "Simpan" untuk menyimpan hasil dan keluar dari aplikasi.
