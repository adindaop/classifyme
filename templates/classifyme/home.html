Tentang Aplikasi

ClassifyMe! adalah aplikasi yang dibuat khusus untuk mengklasifikasikan resensi buku ke dalam resensi positif dan negatif dengan menggunakan teknik preprocessing pada text mining dan Algoritma Naive Bayes.


{% load static %}

Aplikasi ini dibuat oleh Adinda Octadia Putri, mahasiswi Universitas Gunadarma angkatan 2013 sebagai syarat dalam mencapai gelar Sarjana Strata Satu (S1).

Petunjuk Penggunaan

  1. Untuk menggunakan aplikasi, klik tombol "Mulai" yang ada pada bagian paling bawah halaman ini.
  2. Masukkan data buku, pilih training set, dan unggah file testing set berformat .csv dengan delimiter titik koma (;).
  3. Data yang sudah dimasukkan akan tampil. Pilih buku yang akan diproses.
  4. Klik tombol "Proses" di bagian bawah halaman untuk melakukan proses klasifikasi.
  5. Setelah hasil klasifikasi ditampilkan, klik tombol "Selesai".
